StatsBook Tool

This is a tool for error checking WFTDA StatsBooks. It is not in any way official or endorsed by the Women's Flat Track Derby Association.

Note: This tool assumes you are using the manual from the same year as the statsbook file version. So if you feed it a 2018 statsbook, it will check for '/' and 'X' in the lineups, and if you are using a 2019 file, it will check for '-' and '+'
